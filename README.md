> build all assets and live reload your browser.

[![][fly-badge]][fly]
[![][travis-badge]][travis-link]
[![][mit-badge]][mit]

## Install

fly.js
```
npm install --global fly 
```

dev depandency
```
npm install
```

## Usage

### build

```
fly build
```

### launch local server and watch files

```
fly serve
```

### launch local server using built source

```
fly serve_dist
```
